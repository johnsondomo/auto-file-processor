## 📁 Auto File Processor

**Batch file operations made simple.**

```bash
# Rename files
python auto_file_processor.py /path --rename "{index}_{name}"

# Change extensions
python auto_file_processor.py /path --change-ext .txt .md

# Export file list
python auto_file_processor.py /path --export
```

## 🕷️ Web Scraper

**Extract data from any website.**

```bash
# Extract headings
python web_scraper.py -u "https://example.com" -s "h1"

# Extract links
python web_scraper.py -u "https://example.com" -s "a" -a href

# Export to CSV
python web_scraper.py -u "https://shop.com" -s ".price" -o prices.csv
```
